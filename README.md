- ダウンロードされたZIPファイルを展開し、それぞれの環境に合ったZIPを展開
  - windowsユーザの方：MaiMLHandsON_JASIS2025_windows.zip 
  - macユーザの方：MaiMLHandsON_JASIS2025_mac.zip
